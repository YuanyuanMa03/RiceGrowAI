# RG2CH4.py 灌溉功能文档

## 概述

RG2CH4.py文件中的灌溉功能模块主要用于模拟水稻生长过程中的水分管理策略，通过不同的灌溉模式来影响甲烷(CH4)排放和作物生长。该模块提供了5种标准灌溉模式，支持参数化配置，能够适应不同的土壤条件和生长需求。

## 核心函数

### `FillWaterPtn()` - 灌溉模式生成函数

这是灌溉功能的核心函数，负责根据指定的灌溉模式生成相应的水分管理序列。

#### 函数签名
```python
def FillWaterPtn(PintWaterPtn, PintSDur, Sand, 
                 Cali_L=7, BasicFldDays=15, BasicMidDrnDays=3, 
                 BasicRefldDays=10, EndDrnDays=15):
```

#### 参数说明

| 参数名 | 类型 | 默认值 | 描述 |
|--------|------|--------|------|
| PintWaterPtn | int | - | 灌溉模式编号 (1-5) |
| PintSDur | int | - | 总生长期天数 |
| Sand | float | - | 土壤砂含量百分比 |
| Cali_L | int | 7 | 校准参数，影响淹水天数计算 |
| BasicFldDays | int | 15 | 基础淹水天数 |
| BasicMidDrnDays | int | 3 | 基础中期排水天数 |
| BasicRefldDays | int | 10 | 基础复淹天数 |
| EndDrnDays | int | 15 | 末期排水天数 |

#### 返回值

返回一个字典，包含两个键：
- `Regime`: 灌溉阶段序列列表
- `days`: 对应阶段的天数列表

## 灌溉阶段定义

### Regime 类型

| 代码 | 阶段名称 | 描述 |
|------|----------|------|
| 1 | 淹水阶段 (Flooding) | 保持田间淹水状态 |
| 2 | 中期排水阶段 (Mid-season Drainage) | 中期排水，促进根系发育 |
| 3 | 复淹阶段 (Re-flooding) | 排水后重新淹水 |

## 5种标准灌溉模式

### 模式1：标准淹水-排水-复淹模式
- **Regime序列**: `[1, 2, 1, 3, 2]`
- **阶段流程**: 淹水 → 中期排水 → 淹水 → 复淹 → 末期排水
- **特点**: 最完整的灌溉模式，包含所有阶段，适合精细化管理

### 模式2：简化淹水-排水-复淹模式
- **Regime序列**: `[1, 2, 3, 2]`
- **阶段流程**: 淹水 → 中期排水 → 复淹 → 末期排水
- **特点**: 比模式1少一个淹水阶段，适用于中等水分管理需求

### 模式3：基础淹水-复淹模式
- **Regime序列**: `[1, 3, 2]`
- **阶段流程**: 淹水 → 复淹 → 末期排水
- **特点**: 省略中期排水阶段，适合水分需求较高的作物

### 模式4：持续淹水模式
- **Regime序列**: `[1, 2]`
- **阶段流程**: 淹水 → 末期排水
- **特点**: 大部分时间保持淹水状态，适合水生作物

### 模式5：复淹-排水模式
- **Regime序列**: `[3, 2]`
- **阶段流程**: 复淹 → 末期排水
- **特点**: 从复淹阶段开始，省略初始淹水，适合特定生长阶段

## 天数计算逻辑

### 动态调整机制

灌溉模式中的各阶段天数不是固定值，而是根据以下因素动态计算：

1. **总生长期(PintSDur)**: 根据生长期长度优化各阶段分配
2. **土壤砂含量(Sand)**: 砂质土壤排水快，排水天数相应调整
3. **校准参数(Cali_L)**: 影响淹水天数的计算系数

### 计算公式

#### 淹水阶段天数
```python
min(PintSDur, BasicFldDays + Cali_L * round(PintSDur / 40))
```

#### 中期排水阶段天数
```python
min(max(0, PintSDur - i_L), BasicMidDrnDays + int((1 - Sand / 100) * 10))
```

#### 复淹阶段天数
```python
min(max(0, PintSDur - i_L), BasicRefldDays + 3 * round(PintSDur / 40))
```

## 在CH4排放模型中的应用

### 与甲烷排放的关联

灌溉模式通过影响以下因素来调节CH4排放：

1. **氧化还原电位(Eh)**: 不同灌溉阶段影响土壤氧化还原状态
2. **水分含量(WaterC)**: 直接影响甲烷产生和传输
3. **有机质分解**: 水分条件影响有机质分解速率

### 在CH4Flux函数中的调用

在`CH4Flux_day()`和`CH4Flux_coupled()`函数中：

```python
aryWater = FillWaterPtn(IP, DurDate, sand)
WRgm = aryWater['Regime'][w]      # 当前灌溉阶段
WRgmDays = aryWater['days'][w]    # 当前阶段剩余天数
```

### 阶段对Eh值的影响

不同灌溉阶段对氧化还原电位的计算有不同的影响：

```python
if WRgm == 1:  # 淹水阶段
    Eh -= EhvalueD(Eh, -1 * Eh0, 0.125 * (1 - W / Wmax) ** 4 + 0.04, OMNC)
    WaterC = 0.636
elif WRgm == 2:  # 排水阶段
    Eh -= EhvalueD(Eh, EhValueInit, 0.098 * np.exp(-0.6 * CI), 1)
    WaterC -= EhvalueD(WaterC, 0.2, 0.1, 1)
elif WRgm == 3:  # 复淹阶段
    Eh = EhSmthDecrease(Flooded, Eh, EhBase, 20, 0.125 * (1 - W / Wmax) ** 4 + 0.04)
    WaterC = 0.45 + 0.13 - 0.13 * np.random.uniform()
```

## 使用示例

### 基本使用
```python
# 使用模式1，默认参数
aryWater = FillWaterPtn(1, 120, 30)

# 使用模式2，自定义参数
aryWater = FillWaterPtn(2, 100, 25, 
                       BasicFldDays=20,     # 延长基础淹水
                       BasicMidDrnDays=2,   # 缩短中期排水
                       EndDrnDays=10)       # 缩短末期排水
```

### 在CH4排放模拟中使用
```python
# 在CH4排放模拟中调用
result = CH4Flux_day(1, 120, 1, 30, Tair_data, OMS_data, OMN_data, GY_data)
```

## 参数调优建议

### 根据土壤类型调整
- **砂质土壤**: 减少排水天数，增加淹水天数
- **粘质土壤**: 增加排水天数，防止积水

### 根据气候条件调整
- **湿润地区**: 可适当减少淹水天数
- **干旱地区**: 需要增加淹水天数

### 根据作物需求调整
- **水分敏感作物**: 采用模式1或模式2的精细管理
- **耐旱作物**: 可采用模式4的简化管理

## 技术特点

1. **参数化设计**: 所有关键参数都可配置，便于实验和优化
2. **动态计算**: 根据生长期和土壤条件动态调整天数
3. **科学依据**: 基于水稻生长生理和甲烷排放机理
4. **灵活性**: 支持多种灌溉策略，适应不同管理需求

## 相关函数

### 辅助函数
- `EhvalueD()`: 氧化还原电位变化计算
- `FEh()`: 甲烷产生效率函数
- `EhSmthDecrease()`: 氧化还原电位平滑下降函数

### 主要模拟函数
- `CH4Flux_day()`: 独立CH4排放模拟
- `CH4Flux_coupled()`: 与Ricegrow模型耦合的CH4排放模拟

## 总结

RG2CH4.py中的灌溉功能模块提供了一个科学、灵活的水分管理框架，能够有效模拟不同灌溉策略对水稻生长和甲烷排放的影响。通过参数化设计和动态计算机制，该模块可以适应各种土壤、气候和作物条件，为水稻生产的环境影响评估提供有力支持。
